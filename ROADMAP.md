## Performance Targets

### Current (v0.7.0)

- Cold start: ~2-3s
- HMR: ~100-200ms
- Production build: ~5-10s

### Target (v1.0)

- Cold start: < 1s
- HMR: < 50ms
- Production build: < 5s
- Memory usage: < 100MB
